# alarm-clock-timer-cli
A Python command-line alarm clock and timer that alerts the user after a set time.
## Description

This is a simple command-line Alarm Clock / Timer application built with Python. The program allows users to set a countdown timer or an alarm for a specific time, and it alerts the user when the time is reached.

It demonstrates how to use Python’s `time` module along with loops and delays to create time-based functionality. This project helps build an understanding of real-time program execution and basic timing logic.
